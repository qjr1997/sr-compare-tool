# SR Compare Tool - 超分辨率图像对比工具

[![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个专为超分辨率（Super Resolution）训练和效果评估设计的轻量级图像对比工具。支持单张对比和文件夹批量对比，提供多种可视化对比模式，帮助您快速评估超分效果。

![主界面示意图](https://via.placeholder.com/800x400?text=SR+Compare+Tool+Screenshot)

---

## ✨ 主要特性

### 🖼️ 双模式对比
- **单张对比**：手动选择两张图像进行详细对比
- **文件夹对比**：批量加载两个文件夹，自动匹配同名图像对，支持翻页浏览

### 🔍 五种对比模式
| 模式 | 说明 | 快捷键 |
|------|------|--------|
| **并排对比** | 左右并排显示，同步缩放和平移 | `1` |
| **快速切换** | 按住鼠标左键快速切换两张图像 | `2` |
| **滑动对比** | 可拖动的分割线对比 | `3` |
| **局部放大** | 鼠标悬停显示局部100x100区域放大对比 | `4` |
| **差异显示** | 高亮显示两张图像的差异区域 | `5` |

### 📊 自动评估
- **自动PSNR计算**：实时显示峰值信噪比，量化图像相似度
- **自动对齐**：不同尺寸的图像自动对齐到相同大小
- **统一尺寸**：文件夹模式下可设置统一显示尺寸

### ⌨️ 便捷操作
- **方向键翻页**：文件夹模式下使用 `←` `→` 快速切换图像
- **鼠标滚轮缩放**：快速放大缩小查看细节
- **拖拽平移**：按住鼠标拖动查看图像不同区域
- **丰富的快捷键**：支持数字键、功能键等多种快捷操作

## 📖 使用指南

### 单张对比模式

适合对比单组超分结果：

1. 选择**"单张对比"**模式
2. 点击**"加载图1"**选择原始图像（或低分辨率图）
3. 点击**"加载图2"**选择超分后的图像
4. 在下拉菜单选择对比模式（并排/切换/滑动等）
5. 使用鼠标滚轮缩放，拖拽平移查看细节

> 💡 **提示**：即使两张图像尺寸不同，工具也会自动对齐显示

### 文件夹对比模式

适合批量评估超分效果：

1. 选择**"文件夹对比"**模式
2. 点击**"选择文件夹1"**选择包含原图的文件夹
3. 点击**"选择文件夹2"**选择包含超分结果的文件夹
4. 工具会自动：
   - 按文件名（不区分大小写）智能匹配图像对
   - 如果文件名不匹配，则按排序后的一一对应
   - 显示匹配的图像对数量
5. 使用**方向键 `←` `→`** 或 **◀ ▶ 按钮**翻页浏览
6. 可在上方设置**统一尺寸**（如 512x512）确保一致显示

> 💡 **提示**：建议将原图和超分图使用相同的文件名，这样匹配更准确

### 支持的图像格式

- JPEG/JPG (`.jpg`, `.jpeg`)
- PNG (`.png`)
- BMP (`.bmp`)
- TIFF (`.tiff`, `.tif`)
- WebP (`.webp`)
- GIF (`.gif`)

## 📊 PSNR 参考值

PSNR（峰值信噪比）用于量化两张图像的相似度：

| PSNR 值 | 质量评估 |
|---------|----------|
| > 40 dB | 图像质量极好，几乎无差异 |
| 30-40 dB | 图像质量良好，细微差异 |
| 20-30 dB | 图像有可见差异 |
| < 20 dB | 图像差异明显 |

### 技术特点
- **模块化架构**：职责分离，易于维护扩展
- **策略模式**：五种对比模式独立实现，便于添加新模式

## 🐛 常见问题

### Q: 为什么有些图像无法匹配？
**A**: 工具会按文件名（不含扩展名，不区分大小写）进行匹配。如果文件名完全不同，会按字母排序后一一配对。建议保持原图和超分图使用相同文件名。

### Q: 支持 16-bit 图像吗？
**A**: 支持。工具会自动将 16-bit 图像转换为 8-bit 进行显示。

### Q: 可以对比不同尺寸的图像吗？
**A**: 可以。工具会自动将两张图像对齐到相同尺寸显示，保持宽高比并居中放置。

### Q: 如何清理图像缓存？
**A**: 工具使用 LRU 缓存策略自动管理，无需手动清理。
